Über das Projekt

Weather Guesser ist ein interaktives Projekt, bei dem dem Benutzer zufällige Wetterdaten einer Stadt gezeigt werden. Der Spieler muss dann Anhand dieser Daten mithilfe einer Google Maps API die richtige Stadt herausfinden.

Die Idee ist es, eine spielerische und informative Erfahrung zu bieten, indem Benutzer eine zufällige Stadt erraten muss.
